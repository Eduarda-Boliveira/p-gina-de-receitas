# Página de receitas

Uma página de receitas desenvolvida com HTML e CSS como projeto prático para aprimorar conhecimentos em desenvolvimento web.

## Sobre o Projeto

Este projeto consiste em uma página estática de receita, contendo informações como:

- Imagem da receita
- Título e descrição
- Lista de ingredientes
- Modo de preparo
- Layout responsivo

O objetivo foi praticar conceitos fundamentais de HTML semântico e estilização com CSS.

## Tecnologias Utilizadas

- HTML
- CSS
